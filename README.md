# Jarvis Project

A Python-based voice assistant project that can perform various tasks such as playing music and responding to user commands.

## Features

* Voice command support
* Music library integration
* Python-based implementation
* Simple and easy to extend

## Project Structure

* `MAIN.PY` - Main entry point of the application
* `client.py` - Handles client-side functionality
* `musiclibrary.py` - Stores and manages music library data

## Requirements

* Python 3.13+
* Required Python libraries

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Code-EX69/Jarvis-project.git
```

2. Navigate to the project folder

```bash
cd Jarvis-project
```

3. Run the project

```bash
python MAIN.PY
```
